# REJAP

**REJAP** is a gamified Japanese learning platform. Master Hiragana, Katakana, Kanji, and more through interactive quizzes, levels, and a vibrant community leaderboard.

---

## Features

- **Firebase Authentication**: Secure login, registration, and password reset.
- **MongoDB Database**: Stores user data, progress, and audit logs.
- **Admin Dashboard**:
  - View all users
  - Filter/sort/search users by role, lessons completed, username
  - Change user roles (Admin/User)
  - Role-based Access Control: Assign admin/user roles and protect sensitive routes and pages based on user roles
- **Leaderboard**:
  - Ranks users by lessons completed
  - Displays avatars, usernames, medals for top 3
- **Profile Page**:
  - Upload/change avatar
  - View progress per chapter (Hiragana, Katakana, Kanji, Time)
  - Password reset & Logout
- **Chapters/Levels**:
  - Multiple chapters, each with several levels
  - Progress tracking for each chapter
- **Quiz/Lesson System**:
  - Interactive quizzes with HP bar
  - Retry and next level options
  - Progress auto-saved
- **Modern UI**: Responsive, dark-themed, and visually engaging with TailwindCSS, DaisyUI & HyperUI.
- **Cloudinary CDN**: Fast image hosting and delivery for all user and app images.
- **Figma**: UI/UX designed in Figma before development.
- **Vercel**: Frontend hosted on Vercel for fast, global delivery.
- **Render**: Backend hosted on Render for reliability and scalability.

---

## Tech Stack

- **Frontend**: React, Vite, TailwindCSS, DaisyUI, HyperUI & React Router
- **Backend**: Node.js, Express
- **Database**: MongoDB Atlas
- **Authentication**: Firebase Auth
